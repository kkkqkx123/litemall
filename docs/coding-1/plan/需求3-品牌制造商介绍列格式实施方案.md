# 需求3：品牌制造商页面介绍列显示格式实施方案

## 需求描述
商城管理-品牌制造商页面的列表中，介绍列显示格式：第一行"介绍"名称居中显示，第二行及以后方框内内容居左显示。

## 技术方案

### 前端实现

1. **修改表格列渲染**
   ```javascript
   // 在品牌制造商列表组件中
   {
     title: '介绍',
     key: 'desc',
     render: (h, params) => {
       const desc = params.row.desc || '暂无介绍'
       return h('div', {
         style: {
           textAlign: 'center'
         }
       }, [
         h('div', {
           style: {
             fontWeight: 'bold',
             marginBottom: '8px'
           }
         }, '介绍'),
         h('div', {
           style: {
             border: '1px solid #e8eaec',
             padding: '8px',
             textAlign: 'left',
             backgroundColor: '#f8f8f9',
             borderRadius: '4px',
             minHeight: '40px'
           }
         }, desc)
       ])
     }
   }
   ```

2. **使用Element UI组件**
   ```javascript
   // 如果使用Element UI，可以这样实现
   {
     title: '介绍',
     key: 'desc',
     scopedSlots: {
       default: (scope) => {
         const desc = scope.row.desc || '暂无介绍'
         return (
           <div style="text-align: center">
             <div style="font-weight: bold; margin-bottom: 8px">介绍</div>
             <el-card style="text-align: left; min-height: 40px">
               {desc}
             </el-card>
           </div>
         )
       }
     }
   }
   ```

3. **CSS样式方案**
   ```css
   .brand-desc-cell {
     text-align: center;
   }
   
   .brand-desc-title {
     font-weight: bold;
     margin-bottom: 8px;
     color: #303133;
   }
   
   .brand-desc-content {
     border: 1px solid #e8eaec;
     padding: 8px 12px;
     text-align: left;
     background-color: #f8f8f9;
     border-radius: 4px;
     min-height: 40px;
     line-height: 1.5;
   }
   ```

### 组件实现
```javascript
// 创建介绍组件
const BrandDescCell = {
   props: ['desc'],
   template: `
     <div class="brand-desc-cell">
       <div class="brand-desc-title">介绍</div>
       <div class="brand-desc-content">{{ desc || '暂无介绍' }}</div>
     </div>
   `
}
```

## 实施步骤
1. 定位品牌制造商列表页面组件
2. 修改介绍列的render函数或scopedSlots
3. 应用样式实现标题居中、内容居左的格式
4. 确保响应式设计，适配不同屏幕尺寸
5. 测试显示效果

## 预计工作量
- 开发时间：2-3小时
- 测试时间：1小时

## 注意事项
- 考虑长文本的显示和换行处理
- 确保边框样式与整体UI风格一致
- 空值处理，显示默认文案
- 保持表格行高一致
- 考虑响应式适配
