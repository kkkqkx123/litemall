# 功能需求清单
1. 首页新增点击ui后跳转的功能：点击用户数量显示会员管理页面，点击商品数量（或货品数量）显示商品列表页面，点击订单数量显示订单管理页面。
2. 用户管理-会员足迹页面增加删除功能：单个删除功能、多个删除功能。
3. 商城管理-品牌制造商页面的列表中，介绍列显示格式：第一行“介绍”名称居中显示，第二行及以后方框内内容居左显示。
4. 统计报表-订单统计页面中，订单统计功能拓展：按日、月、季、年统计，按商品类别统计（默认显示所有类别）。
5. 系统管理-操作日志页面中，针对该页面的表格增加按列查询功能：按操作时间查询、按操作状态查询。
6. 商品管理-商品评论页面中，实现列表及搜索功能调整：
    - 增加商品名称、商品类别列，删除评论图片列。
    - 时间列仅显示年月日。
    - 合并搜索输入框，支持根据用户ID或商品名称（模糊搜索）查询。
    - 输入用户ID搜索，显示该用户给所有商品的打分记录。
    - 输入商品名称搜索，显示所有给该商品打分的用户记录。
    - 打分列增加排序功能，点击可切换排序方式。
7. 商品打分统计功能（前端+后端）：
    - 初始列表显示商品ID、商品名称、商品类别、打分平均分（支持按平均分排序）、打分用户数。
    - 搜索功能：选择商品类别，按打分平均分高低显示该类别下商品。
8. 商品评论词云图（前端+后端）。
